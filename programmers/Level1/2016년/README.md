# [프로그래머스 Level1] 2016년
- https://programmers.co.kr/learn/courses/30/lessons/12901
- 요일을 계산해서 반환하는 문제이다.2가지 방법이 있다.
  1. 계산을 해서 푸는 방법
    - 총 일수를 계산한 뒤 7로 나누면 요일을 구할 수 있다.
      - 이 때 윤년을 고려하여 총일수를 구한다.
  2. Date객체를 이용해 푸는 방법
    - Date객체를 생성하고, Date의 문자열을slice하여 요일만 추출한다.
- 가장 빠른 방법은 직접 계산을 해서 푸는 방법이었다. 평균적으로 0.06ms 의 속도로 풀 수 있었다.
  - 반면 2번의 방법은 1번에 방법에 최소 5배~50배까지 느렸다. 속도차이가 균일했던 1번에 비해 2번은 속도차이도 편차가 심했다.
- 이 문제를 통해 JS에서 요일을 다루는 방법을 생각하게 되었다. 특히 무작정 Date객체를 사용하기 보다는 문제에 맞게 최적화된 방법으로 알고리즘을 푸는게 더 도움될 때가 있다는걸 이 문제를 통해 알게되었다.
  - 즉 무작정 편리하다고 사용하기보다는 알고리즘으로 간단하게 구현할 수 있다면 직접구현하는게 더 빠를 수 있다는 걸 느꼈다.
