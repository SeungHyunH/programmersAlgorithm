# [프로그래머스 Level2] H-index
- https://programmers.co.kr/learn/courses/30/lessons/42747
- 정렬해서 주어진 조건에 맞는 h-index의 최대값을 구하는 문제다.
  - 문제의 설명이 많이 이상한 문제다. 코드설명만 보면 요소를 기준으로 찾는걸로 설명이 되어있다. 예시 또한 그런것처럼 설명이 되어 있다. 하지만 이 문제를 풀기 위해서는 요소를 기준으로 하는것이 아니라 순서를 기준으로 해야한다. 예를들어 [4,5,6,7] -> 이렇게 있다면 5를 기준으로 하는것이 아니라 5의 순서인 2번째 즉 2를 기준으로 나눠야 한다.
- 어렵진 않았지만 좀 설명이 이상한 문제였다.실제로 문제에도 위키링크가 따로 있는걸 보면 논란이 좀 많은 문제였던듯 하다.
