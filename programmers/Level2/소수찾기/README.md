# [프로그래머스 Level2] 소수찾기
- https://programmers.co.kr/learn/courses/30/lessons/42839
- 레벨1에서 풀었던 소수찾기(https://programmers.co.kr/learn/courses/30/lessons/12921)의 업그레드 버전, 배열의 숫자를 조합해 숫자를 만들어 소수를 판별하는 문제이다.
  - 완전탐색을 하는 문제이다. 여기서 핵심은 처음 주어지는 numbers를 중복해서 사용하면 안되는 부분이다.
    - 여기서 중복을 피하기 위해 푸는 코드에서 배열의 deep copy가 필요했고 그 과정에서 [...] 의 코드를 발견했다. 이렇게 하면 deep copy해서 배열을 반환해준다는 것을 알았다.
- 이 문제는 deep copy를 어떻게 하느냐를 몰라서 한참 해맸던 문제였다. 다시 한 번 풀어볼만한 문제인 것 같다. 몰랐던걸 알게되서 좋았던 문제였다.
